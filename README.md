# Folder Timeline

A macOS Quick Action that turns any folder into an interactive timeline chart —
files and sub-folders plotted by date, grouped by year, quarter, month, or day.

## Install

### Option 1 — Download the zip (no Terminal needed)
1. Download **FolderTimeline.zip** from the [latest release](../../releases/latest)
2. Extract it and double-click **Folder Timeline Setup.app**
3. Terminal opens and walks you through the rest

> **"App can't be opened" (Gatekeeper)?** Right-click the app → Open → Open.
> One-time step for apps downloaded from the internet.

### Option 2 — Clone and run (recommended, zero friction)
```bash
git clone https://github.com/lancel00zz/FolderTimeline.git
cd FolderTimeline
bash setup.command
```

## Use
Right-click any folder in Finder → Quick Actions → **Folder Timeline**.
Click a bar to open items. Filter by name. Press Escape to close.

## Uninstall
Run `bash setup.command` and choose option 2.
