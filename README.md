# Folder Timeline

A macOS Quick Action that turns any folder into an interactive timeline chart —
files and sub-folders plotted by date, grouped by year, quarter, month, or day.

## Install or Uninstall
Double-click **Folder Timeline Setup.app** — it opens Terminal and asks:

    1. Install Folder Timeline
    2. Uninstall Folder Timeline

Requires macOS 12+ and an internet connection on first install.

> **"App can't be opened" (Gatekeeper)?** Right-click → Open → Open.
> One-time step for apps downloaded from the internet.

> **Prefer Terminal?** Run `bash setup.command` directly.

## Use
Right-click any folder in Finder → Quick Actions → **Folder Timeline**.
Click a bar to open items. Filter by name. Press Escape to close.
